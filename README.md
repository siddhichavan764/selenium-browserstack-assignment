# BrowserStack Selenium Assignment

This project scrapes the first 5 opinion articles from El País using Selenium.

Features:
- Extract article titles
- Translate Spanish titles to English
- Extract article content
- Download article images
- Analyze repeated words in titles

Technologies Used:
- Python
- Selenium
- Requests
- Googletrans

Run the script:

pip install -r requirements.txt
python scraper.py
